**选择排序**

将一串数字按照从小到大排序； 
 
具有下面的特点：
1. 采用线性查找（按照数组的顺序依次查找），找出所有数字中最小的一个；
2. 将最小的数字放到最左边；
3. 然后比较剩下的数字，再找出最小的一个；
4. 放在第二个位置；
5. 依次循环;
6. 第一次比较n-1次，第二次比较n-2次，最终的复杂度与冒泡排序一样O(n^2);

***

**代码**
```
function choose(arr) {
  let temp = arr.slice();
  let index = 0; // 用来储存最小值的索引
  let min = 0;
  for (var i=0, len=temp.length; i < len; i++) {
    min = temp[i];
    index = i;
    for (let j = i; j < len; j++) {
      if (temp[j] < min) {
        min = temp[j];
        index = j;
      }
    }
    // 将最小的那个值放到最前面
    let number = temp[index];
    temp.splice(index, 1);
    temp.unshift(number);
  }
  // 输出的时候采用倒序的方式
  return temp.reverse();
}

let arr = [1, 3, 21, 2, 46, 3, 4, 5, 7, 8, 90];
console.log(choose(arr))
```
